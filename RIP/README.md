# 🌐 RIP Routing Lab

## 🎯 Objective
Implement dynamic routing using RIP v2 between routers.

## 🧠 Summary
RIP is a simple distance-vector protocol that uses hop count as a metric. It’s easy to set up but not suitable for large networks.
